# Lumina Paws

Premium American pet companion app for **dogs** and **royal cats**.

## Features
- Mobile-app feel (bottom nav, glass UI)
- Dog mode (warm gold) + Cat royal mode (indigo/amethyst)
- Charity messaging (80-85% to rescues)
- Creative care bots: Pawlo & Lady Whisk
- Reduced coin economy
- Admin dashboard at `/admin`

## Run
```bash
npm install
npm run dev
```
